# Architecture

Vendor Replacement Intelligence is a static-friendly TypeScript executive-intelligence surface.

## Layers

1. `src/data/sampleVendorReplacement.ts`
   Synthetic vendor-estate review records across AI, identity, CRM, analytics, martech, workflow, and support categories.
2. `src/analyze.ts`
   Computes replacement score, blocked-exit counts, modeled savings totals, and board-readable findings.
3. `src/services/vendorReplacementIntelligenceService.ts`
   Shapes the analysis into overview metrics, replacement lanes, switching risks, savings narrative packets, and board memo outputs.
4. `src/services/render.ts`
   Produces the executive HTML surfaces.
5. `scripts/prerender.ts`
   Exports the static site and JSON payloads for GitHub Pages.
